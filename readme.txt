Add custom title, show title, show content of any selected page in a widget ready area

== Description ==

Show content of any selected page in a widget.

1. In the back office, drag and drop the widget on widget zone like you want.
2. Put a custom title, show or not the custom title.
3. Choose the page you want in the select box.
4. Show or not the page title, the page content, the page excerpt.
5. This widget run multiple instances, so you can put any page you want in different widget zone.

If you use it please rate it.

If you use it please don't forget compatibility box.


== Example of usage == 

* useful for welcome message in your homepage
* useful for about stuff in your homepage
* useful for address information in your homepage
* useful for a lot of things


== Installation ==
  
1. Download the .zip file
2. Extract the contents and upload into your wp-content/plugins directory
3. Activate the plugin/widget in your WordPress Dashboard
